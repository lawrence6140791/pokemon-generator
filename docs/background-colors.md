## 對比初步檢查
使用近似 WCAG 公式估計。實際值建議後續以工具（例如 axe, Lighthouse）驗證。

## 可改善項目
- psychic (#F85888) 與 #222 對比在某些螢幕可能不足，可在 hover/focus 期間加細邊框 (#222) 或改深一階色。
- 若使用者啟用高對比模式可切換至無漸層純色並調整字色。

## 後續建議
1. 加入 prefers-contrast CSS 媒體查詢（未來瀏覽器支援時）。
2. 測試深色模式：需另行定義 dark theme palette。
3. 對 badge 增加 `aria-label="type: fire"` 或本地語系化文字。
